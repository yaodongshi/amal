Key Features

  • Generate Lot No Based on Configuration Like Today date/Purchase Order Date/Schedule Date of Incoming Shipment.
  • If we have Purchase same product second time so add Sequence no after the Lot no.
  • Lot no and Done qty automaticall set in Incoming Shipment when we have confirm purchase order.
  • Lot no Generate After Confirmation of Purchase Order.
  • Generate Lot no Format YYYYMMDD
  • Same Lot no Use in Backorder.If It is Created.
  • Tracking By Lots Must be set in the Product.
  • Auto Generate Lot Configuration.
    Go to Inventory -> Operation Types -> Select Receipt Select the Existing Lots If not checked
  • Go to Purchase Order -> Configuration -> Settings.
  • Lot No Generate Based on Purchase Order Date Create Purchase Order and Confirm It.
